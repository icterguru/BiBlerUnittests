Class EntryDict

source code

The exchange format of entries. It is a dictionary where all EntryListColumn fields are predefined keys that cannot be removed.

>>> d = EntryDict()
>>> print d
    {'id': '', ...}
>>> del d[EntryListColumn.ID]
>>> print d
    {'id': '', ...}
